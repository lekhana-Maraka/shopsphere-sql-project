# 🛒 ShopSphere SQL Project

## 📌 Project Overview

ShopSphere is an E-Commerce Database Management System developed using **MySQL Workbench**. The project demonstrates database design, table relationships, CRUD operations, business queries, joins, subqueries, views, stored procedures, triggers, transactions, and other advanced SQL concepts.
